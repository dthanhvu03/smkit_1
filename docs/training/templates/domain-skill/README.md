# Domain skill scaffold

1. Copy this folder to `engine/skills/<your-id>/`.
2. Replace `DOMAIN_ID` and placeholders in `SKILL.md`, `skill.kit.yaml`, `references/domain-guide.md`.
3. Wire routing (see [../../03-domain-skill-guide.md](../../03-domain-skill-guide.md)).
4. `smkit build` · `smkit doctor` · dogfood one task.

Do not commit placeholder `DOMAIN_ID` into the shipped kit engine — this tree is a **template only**.
